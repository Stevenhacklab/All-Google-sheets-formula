Google Sheets Formula Reference


Formulas Usable in Both Cells and Conditional Formatting

Math

Formula NameDescriptionExample
MODReturns the remainder after division=MOD(ROW(), 2)=0

Logic

Formula NameDescriptionExample
IFReturns one value if a condition is true, another if false=IF(A1>10, "Yes", "No")
ANDReturns TRUE if all conditions are true=AND(A1>0, B1<10)
ORReturns TRUE if any condition is true=OR(A1=5, B1=10)
NOTReturns the opposite of a logical value=NOT(A1=5)

Formulas Usable Only in Cells

Lookup

Formula NameDescriptionExample
VLOOKUPSearches the first column for a key and returns a value in the same row=VLOOKUP("Apple", A2:B10, 2, FALSE)
HLOOKUPSearches the first row for a key and returns a value in the same column=HLOOKUP("Q1", A1:D2, 2, FALSE)

Data Import

Formula NameDescriptionExample
IMPORTRANGEImports data from another spreadsheet=IMPORTRANGE("sheet_url", "Sheet1!A1:B10")

Array

Formula NameDescriptionExample
ARRAYFORMULAPerforms a calculation over a range=ARRAYFORMULA(A1:A10*2)
UNIQUEReturns unique rows in the provided source range=UNIQUE(A1:A10)
